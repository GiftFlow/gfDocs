set_opened

public void set_opened()

Mutates $this->messages objects with unopened/opened info. Checks to see if any of the messages in $this->messages are unread. This is done by checking the message_deliveries table for rows with matching message and user IDs. If they are unread, the opened property is set to FALSE.
