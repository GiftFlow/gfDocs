Class Notify

Notify

public class Notify

The Notify library handles all notifications and alerts.

Notifications are very complex and thus this class deals with two different kinds of notifications:

1. Alerts

These are email alerts which are sent when an event takes place.

2. Notification feeds

This is a feed of all the latest events relevant to a user. This is very similar to Facebook's notification feed system.
