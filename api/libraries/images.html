public string $image_library = 'GD2'

Default image library. I highly recommend changing this to imagemagick, because its performance is *way* better than GD2.

If you set $config['image_library'] in a config file, it will override the default value for this variable.

square

public mixed square(string originalFile, string newFile, integer offset)

Crop an image so that it becomes square. I fyou need a square thumbnail, run this method first and resize afterwards.

By default the original image is cropped and overwritten, but you can supply a destination path if you wish to retain the original image and create a new, cropped version of that image somewhere else.

By default the image is cropped from the center, but you can supply an optional offset parameter.

Parameters:
originalFile - The full path and filename of the image to be cropped
newFile - (optional) The full destination path and filename
offset - (optional) Offset for either x or y axis
Returns:
False if no action was performed, else void

public void squareThumb(mixed originalFile, mixed newFile, int newSize, integer offset, $originalFile Full, $newFile The, $newSize (optional))

Calls $this->square and $this->resize. The squared image is always a copy of the original, since it is a thumbnail.
