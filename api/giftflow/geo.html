geocode

public object geocode(string address, object object)

Convert an address into a lat/lng pair Optionally, an object (such as a DataMapper model object, can be passed to the method. It then assigns the geocoding results as properties of this object instead of creating and then returning a new object.

Documentation Here: http://code.google.com/apis/maps/documentation/geocoding

Parameters:
address - Location keyword / address to encode
object - Object to store results in (optional)
