public void csv_export(mixed object, mixed filename, str fields, bool include_header)

Convert a DataMapper model into an associative array.

Param DataMapper:
$object The DataMapper Object to export
Param mixed:
filename The filename to export to, or a file pointer. If this is a file pointer, it will not be closed.
Param array:
$fields Array of fields to include. If empty, includes all database columns.
Param bool:
$include_header If FALSE the header is not exported with the CSV. Not recommended if planning to import this data.
Return bool:
TRUE on success, or FALSE on failure.

csv_import

public void csv_import(mixed object, mixed filename, str fields, bool header_row, mixed callback)

Import objects from a CSV file.

Completely empty rows are automatically skipped, as are rows that start with a # sign (assumed to be comments).

Param DataMapper:
$object The type of DataMapper Object to import
Param mixed:
$filename Name of CSV file, or a file pointer.
$callback A callback method for each row. Can return FALSE on failure to save, or 'stop' to stop the import.
Param array:
$fields If empty, the database fields are used. Otherwise used to limit what fields are saved.
Param boolean:
$header_row If true, the first line is assumed to be a header row. Defaults to true.
Return array:
Array of imported objects, or FALSE if unable to import.
