__construct

public void __construct(string request, string method, mixed params, string body, string uri, string parameters, array headers)

Construct the request to be signed. Parses or appends the parameters in the params url. When you supply an params array, then the params should not be urlencoded. When you supply a string, then it is assumed it is of the type application/x-www-form-urlencoded

Parameters:
request - url
method - PUT, GET, POST etc.
params - string (for urlencoded data, or array with name/value pairs)
body - optional body for PUT and/or POST requests

sign

public void sign(int usr_id, array secrets, string name)

Sign our message in the way the server understands. Set the needed oauth_xxxx parameters.

Parameters:
usr_id - (optional) user that wants to sign this request
secrets - secrets used for signing, when empty then secrets will be fetched from the token registry
name - name of the token to be used for signing
Exception:
OAuthException when there is no oauth relation with the server
OAuthException when we don't support the signing methods of the server
