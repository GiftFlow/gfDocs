public class Transaction
extends DataMapperExtension

ABOUT THIS EXTENSION

The purpose of this extension is to alter the order in which DataMapper deletes objects. Originally, the object was deleted and then its relationships. This caused problems with the foreign key constraints of our database, and thus this extention reverses the process: the relationships are deleted first, followed by the object itself.
