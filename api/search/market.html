give

public boolean give(array options)

User requests to give his gift to another user

$options array's arguments same as $single_demand, except for the recipient ID. In the request use cases, the user ID is that of the demander and the user ID of the other transaction participant can be derived from the user ID of the requested good's owner. In the give and request cases, however, the Demander ID and Owner ID are the same. Thus we need to provide the ID of the recipient. It will overwrite the user_id option.

Arg:
int recipient_id ID of recipient
