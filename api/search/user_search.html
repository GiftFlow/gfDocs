_geosearch_clauses

protected boolean _geosearch_clauses(object options, float options->latitude, float options->longitude, float options->address)

Adds where clauses that limit results to a specific geographic area

Calculates bounds by determining how many degrees of latitude and longitude the search radius encompasses. This is not a complete solution, however, since the shape of the bounds is a square, not a circle. Thus to be truly accurate another where clause limiting results by the calculated distance field should be used.

Parameters:
options - Location object
options->latitude - Latitude
options->longitude - Longitude
options->address - Formatted address
